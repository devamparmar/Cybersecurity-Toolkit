#  Cybersecurity Toolkit

A simple cybersecurity toolkit developed using Python and deployed with GitHub Pages.

##  Overview

Cybersecurity Toolkit is designed to help users improve their online security through essential security checks. The project provides tools for analyzing password strength and checking URL safety.

##  Features

###  Password Strength Checker

* Analyzes password strength
* Identifies weak, medium, and strong passwords
* Encourages secure password creation

###  URL Safety Checker

* Checks whether a URL is safe or suspicious
* Detects non-HTTPS links
* Helps users avoid potentially unsafe websites

##  Technologies Used

* CSS
* JavaScript
* GitHub Pages
