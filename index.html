Cybersecurity Toolkit

Password Strength Checker | URL Safety Checker | File Encryption & Decryption

Project Overview

Cybersecurity Toolkit is a simple Python-based security project designed to improve cybersecurity awareness. The toolkit includes tools for checking password strength, analyzing URL safety, and encrypting or decrypting text securely.

Password Strength Checker

This feature checks whether a password is Weak, Medium, or Strong based on length, numbers, uppercase letters, and special characters.

URL Safety Checker

This feature analyzes URLs and helps identify suspicious or potentially unsafe links.

File Encryption & Decryption

This feature encrypts text to protect information and decrypts it back to its original form when needed.

Key Features

  • Password Security Analysis
  • URL Safety Verification
  • Text Encryption
  • Text Decryption
  • Simple and User Friendly Interface
